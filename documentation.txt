Lingo.js

instantiating a new Lingo class:

var lingo = new Lingo() - will create a Lingo class ready to use.
Parameters

Required:

Element:

  var lingo = new Lingo('.main-div')
  This will set the root div for your LingoJS app.
Models:
  var lingo = new Lingo('.s',{
    Models:{
    //Set your models here
      Model:'ModelValue',
      //This is also okay
      'L-Model':'ModelValue'
  }
});
  

Attributes:
To set master and slave elements add attributes to your elements like this.

<input lm="Model"> <-Master
<p lv="Model">Slave</p>

Master will update the Models object onchange.

-n attribute:
The -n attribute will add the Network jobs to the element such as simple AJAX REST and more.
